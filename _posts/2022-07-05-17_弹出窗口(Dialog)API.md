## [K.dialog(options)](http://kindeditor.net/docs/dialog.html#id2)

创建弹出窗口。

-   -   参数:

        object options: 配置信息

-   返回: KDialog

-   继承: KWidget ( [K.widget(options)](http://kindeditor.net/docs/widget.html#k-widget) )

示例:

```
var dialog = K.dialog({
        width : 500,
        title : '测试窗口',
        body : '<div style="margin:10px;"><strong>内容</strong></div>',
        closeBtn : {
                name : '关闭',
                click : function(e) {
                        dialog.remove();
                }
        },
        yesBtn : {
                name : '确定',
                click : function(e) {
                        alert(this.value);
                }
        },
        noBtn : {
                name : '取消',
                click : function(e) {
                        dialog.remove();
                }
        }
});
```
